Help and Info


To play, search a Haikyuu!! character to start guessing!


Red is incorrect.

Blue means the correct value is either higher (▲) or lower (▼)

Green is correct

If a number shows up as -1 then that character is either a manager or a coach.

If a number shows up as 4 then that character is an adult that is a professional volleyball player. Certain characters have both their high school version and their timeskip version.

After 5 guesses the character list will be sorted by position.

After 8 guesses the character list will be sorted by school.


Character information was taken from the Haikyuu Wiki.

Only characters who had information for all categories are included.
